Disclaimer and Notification Manager for Authors

== Description ==

This plugin simplify the process of adding Disclaimers and Custom Notifications at the top, middle or bottom of articles written by Guest contributors.

= Features =

1. Set a disclaimer text or notification that will display in every post made by guest authors.

2. Choose the authors that will have a disclaimer or notification displayed in their post.

3. Ability to set custom disclaimer or notification in each and every post.

4. Built-in editor for adding CSS Styles for the Disclaimer text / notification

5. Choose which post position to display the Disclaimer / custom notification - top, middle, bottom.

6. Placeholder to auto-magically include authors first name, last name and username without to manually type the author name of the article..
